# Excel Sales Analysis

## Project Objective
This project analyzes product sales performance using Excel.

## Tools Used
- Excel
- Pivot Tables
- Charts

## Dataset
Sales dataset containing information about:
- Order Date
- Product
- Region
- Salesperson
- Quantity
- Price per Unit
- Sales
- Target

## Analysis Performed
- Sales aggregation using Pivot Tables
- Product performance analysis
- Visualization using Excel charts

## Key Insights
- Top Product: Printer
- Total Sales: 281M
- Sales performance comparison across products
